# Streamlit CO₂ Emissions App

This demo shows how to visualize monthly CO₂ emissions using a Streamlit app.

1. Install dependencies: `pip install streamlit pandas altair openpyxl`.
2. Run the app: `streamlit run co2_emissions_app.py`.
3. Upload an Excel file containing `Date` and `CO2` columns.
4. View the live chart and download the uploaded data using the provided button.
